Мощный C++ веб-сервер

Crow - современный, быстрый и легковесный фреймворк для создания веб-приложений на C++. Идеально подходит для высоконагруженных систем и микросервисов.

server.cpp
#include "crow.h"

int main() {
    crow::SimpleApp app;
    
    // Простой эндпоинт
    CROW_ROUTE(app, "/")([](){
        return "Привет, мир!";
    });
    
    // JSON API
    CROW_ROUTE(app, "/json")
    ([]{
        crow::json::wvalue x;
        x["message"] = "Привет из C++!";
        x["status"] = 200;
        return x;
    });
    
    // Запуск сервера
    app.port(8080).multithreaded().run();
}

Почему Crow?

Современные возможности для создания высокопроизводительных веб-приложений на C++

Молниеносная скорость

Обрабатывайте до 150,000 запросов в секунду благодаря асинхронному вводу-выводу и эффективности C++.

Простой API

Интуитивно понятный синтаксис для маршрутизации, работы с JSON и обработки запросов.

Минимальные зависимости

Только Boost.Asio - никаких тяжелых фреймворков. Легковесный и простой в интеграции.

Middleware
// Аутентификация через Middleware
struct AuthMiddleware {
    struct context {
        User user;
        bool is_authenticated = false;
    };
    
    void before_handle(crow::request& req, 
                      crow::response& res, 
                      context& ctx) {
        auto token = req.get_header_value("Auth-Token");
        if (auto user = auth_service.verify(token)) {
            ctx.user = *user;
            ctx.is_authenticated = true;
        } else {
            res.code = 401;
            res.end();
        }
    }
    
    void after_handle(crow::request& req, 
                     crow::response& res, 
                     context& ctx) {}
};
